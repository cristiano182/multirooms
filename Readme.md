## Multirooms

Simple lib for create multirooms chat

## About this Project

Make an scaleable chat.Handle location and online users.

## Getting Started

$ npm install multirooms --save

or

$ yarn add multirooms --save


## How to use

Mongoose and Socket.io is a dependency for this library.
